## Agent skills

### Issue tracker

Issues 以 Local markdown 文件形式存放在 `.scratch/<feature-slug>/` 下。详见 `docs/agents/issue-tracker.md`。

### Triage labels

使用默认标签名称（`needs-triage`、`needs-info`、`ready-for-agent`、`ready-for-human`、`wontfix`）。详见 `docs/agents/triage-labels.md`。

### Domain docs

Single-context 布局，一个 `CONTEXT.md` + `docs/adr/` 位于项目根目录。详见 `docs/agents/domain.md`。
